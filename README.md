# Vimophone

A Vim ideology based music instrument

Checkout right now at https://vimophone.pages.dev

## Controls
Keys **`C D E F G A B`** represent their respective notes

Modifiers -
- `,` for +2 Octave
- `K` for -1 Octave
- `I` for +1 Octave
- `J` for -1 transpose (flat)
- `U` for +1 transpose (sharp)
